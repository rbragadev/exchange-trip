# Foundation Notes

This repository is a foundation-only scaffold with no business domain yet.

- API: NestJS only with `/api/health` and `/api/app-info`
- Admin: basic SaaS shell with login/dashboard/foundation/settings pages
- Mobile: Expo screens for home, API status, settings and fluxo
- Data: Prisma project ready with an intentionally empty seed
- Infra: PostgreSQL via docker-compose
